# babynote
유치원 어플

키즈노트를 모티브해서 만들어본 개인 프로젝트 입니다.


Android , Kotlin , Node.js , MySQL , Retrofit2를 사용하였습니다.


서버 깃헙

https://github.com/rain2353/babynote_server
